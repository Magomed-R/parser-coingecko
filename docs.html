Язык: Python
База данных: PostreSQL

Подготовка к запуску

Перед тем, как запускать ботов, необходимо:
  1. Установить Python, если он ещё не установлен
  2. Установить PostreSQL
  3. Желательно установить Excel
  4. Открыть терминал и перейти в папку с проектом (если нужна более подробная инструкция, напишите мне об этом)
  5. Создать виртуальную среду в папке командой python -m venv .venv
  6. Активировать виртуальную среду первой ИЛИ второй командой, в зависимости от вашего терминала
                # В cmd.exe
            .venv\Scripts\activate.bat
            # В PowerShell
            .venv\Scripts\Activate.ps1
  7. Установить все необходимые зависимости командой pip install -r requirements.txt

Запуск ботов

Проект состоит из одного полного парсера и нескольких парсеров конкретных монет. Парсеров конкретных монет может быть сколько угодно. Каждого бота необходимо запускать отдельным процессом. Сделано это для более скоростной работы, так как ни асинхронность, ни многопоточность не дают такого увеличения скорости, как разделение на процессы
Для запуска ботов необходимо:
  1. Настроить конфигурацию полного парсера (название файла обязательно должно быть fullparser.ini)
  2. Настроить конфигурацию частичных парсеров(название файла конфигурации может быть любым, главное, чтобы имело расширение .ini и имело все соответствующие поля)
  3. Запустить полный парсер командой python src/full_parsing.py (если монеты парсятся без ошибок и появляются в БД, то переходить к следующему шагу)
  4. Открыть другой терминал или вторую вкладку терминала, снова перейти в папку с проеком и снова активировать виртуальную среду
  5. Запустить частичные парсеры командой python src/module.py, после чего потребуется ввести путь до конфига. Например module-1.ini
Готово! Боты работают, монеты парсятся!
